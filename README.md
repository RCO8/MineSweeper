# MineSweeper
pygame라이브러리를 이용한 간단한 지뢰찾기 게임

사용한 개발환경 : Pycharm
사용한 라이브러리 : pygame

게임 구조
------------------------------
지뢰판 아래에 각 속성들이 있다.

1. 리플레이 버튼 : 게임을 초기화한다.

2. 너비 및 높이 조절 : 각 길이를 조절 후 리셋하면 길이가 바뀐다.

3. 지뢰수 조절 : 2번의 옵션이랑 비슷하나 지뢰 개수를 조절한다.
